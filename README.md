# Job List Project
List Job Project with Vue3 and Vuex state management
For online app : https://jobsearch-vue.netlify.app/

## Routing
```
/ => Home Page
/jobList => listing jobs
/jobDetail:id => Job detail page for selected job from joblist

## Project setup
```
npm install
```
